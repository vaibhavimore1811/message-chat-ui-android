# 📖 Overview

**Message Chat UI Android** is a modern Android messaging interface project developed using Java.

This project focuses on creating a premium chat application UI with futuristic animations, smooth messaging transitions, Glassmorphism effects, and AI-inspired user experience.

The application showcases:

- 💬 Modern Chat Screens
- ✨ Animated Message Bubbles
- 🎨 Glassmorphism Chat Design
- ⚡ Smooth RecyclerView Animations
- 🤖 AI Chat Style Interface
- 🌌 Neon & Cyberpunk Aesthetics
- 📱 Responsive Android Layouts

Perfect for:

- Android Developers
- UI/UX Designers
- Chat Application Templates
- Portfolio Projects
- Open Source Learning

# ⚙️ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/yourusername/message-chat-ui-android.git
```

---

## 2️⃣ Open in Android Studio

Recommended:

- Android Studio Giraffe or newer

---

## 3️⃣ Sync Gradle

```bash
File → Sync Project with Gradle Files
```

---

## 4️⃣ Run Application

- Connect Android device/emulator
- Click ▶ Run
